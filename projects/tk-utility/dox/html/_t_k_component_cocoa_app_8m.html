NSString* const TKComponentCocoaAppShouldUseRelativePathsKey = "TKComponentCocoaAppShouldUseRelativePaths"

Indicate whether we should interpret all paths (with the exception of the application and support files) as relative to application path.

If yes, any given paths will be appended to the application path and then expanded. Relative paths like: ../../DATA (DATA folder two levels up from the application path).
